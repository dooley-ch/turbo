# Using The Paradox Engine

## Introduction

Unfortunately, I was unable to locate any documentaiton on the Borland Paradox Engine 3.0 and 
unfortunately the install disks are also incomplete!  However it is possible to install the TP 
version of the code from the disks.  What follows is based on my analysis of the demo programs
included on the disks.
